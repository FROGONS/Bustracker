Desenvolvimento de Aplicações Web

Projeto de BusTracker

Criar uma aplicação web responsiva para o monitoramento do transporte público de Belo Horizonte

2026 Ano
Autores Arthur David, Arthur Lara, Kaua Vitor, Lucas Neiss, Sofia e Rafael Valdivio

Proposta

Desenvolver uma aplicação web que, com base na API da Prefeitura de Belo Horizonte, forneça aos usuários uma clara visualização dos ônibus públicos na cidade

Sobre o projeto

Requesitos

Para a implementação da página web, várias demandas foram requeridas

  • Filtragem por linha
  • Marcação de ponto mais próximo
  • Utilização de frameworks (Bootstrap) e bibliotecas(Leaflet)
  • Adaptabilidade a difentes telas

O Desafio

Obstáculos reais enfrentados durante o desenvolvimento do Rastreador de Ônibus de BH.

Principais Dificuldades

Bloqueio de CORS & Rede

A API pública bloqueia requisições externas diretas com erro de CORS. Proxies públicos da internet falharam por timeout ou instabilidade, inviabilizando o consumo direto pelo navegador na rede escolar.

Ambiguidade de Siglas

A API carecia de documentação clara. Chaves como EV (Empresa) causavam colisão de dados, ocultando os veículos. Foi necessário decodificar que NV identificava o veículo físico e NL a linha real para o cruzamento com os CSVs.

Conversão de Coordenadas UTM

As paradas de ônibus nos arquivos locais estavam no formato POINT(X Y) sob a projeção UTM SIRGAS 2000 — incompatível com mapas web. Foi preciso integrar a biblioteca Proj4js para convertê-las em Lat/Lng em tempo real.

Nossa Solução

Como resolvemos o problema?

Arquitetura Híbrida Inteligente: Desenvolvemos um servidor Proxy leve em Node.js customizado para rodar localmente. Ele intercepta as requisições, contorna o bloqueio de CORS da BHTRANS e entrega os dados limpos e validados para o mapa.

Processamento em Memória: O front-end faz uso de coleções do tipo Map para indexação instantânea dos CSVs e implementa uma lógica inteligente de Linha Base, permitindo isolar no mapa uma linha específica e todas as suas sub-linhas (ex: agrupar 67-01 e 67-02 ao pesquisar por "67").
